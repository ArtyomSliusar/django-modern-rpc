=================
Django-modern-rpc
=================

Overview
--------

Django-modern-rpc provides a simple solution to implement a remote procedure call (RPC) server as part of your Django
project. It supports all major Django (from 1.8 to 1.10) and Python (2.7 to 3.5) versions.

.. note::
   This project is under development, and should not yet be used in a production environment.

The main features of the project are:

- XML-RPC & JSON-RPC support
- :ref:`Pre-defined exceptions <exceptions>` to easily and correctly return common errors to clients.
- :ref:`Multiple entry points declaration <multiple_entry_points>`
